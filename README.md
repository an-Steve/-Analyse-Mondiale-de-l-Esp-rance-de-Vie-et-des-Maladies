## →  Description

Ce projet  présente une analyse interactive et complète de l'espérance de vie à l'échelle mondiale, en étudiant les facteurs de santé publique qui l'influencent. 
Ce projet utilise le dataset **"Life Expectancy Data GHO"**, disponible sur Kaggle via ce lien : `[https://www.kaggle.com/datasets/adamsmith852/life-expectancy-data-gho](https://www.kaggle.com/datasets/adamsmith852/life-expectancy-data-gho).`

À travers `Tableau Public`, ce projet explore plusieurs dashboards dynamiques les liens entre l'espérance de vie, les maladies infectieuses, les comportements à risque et les disparités de genre à travers le monde.

---

## → Objectifs

- **Analyser** les tendances de l'espérance de vie par pays et par genre
- **Visualiser** l'impact des maladies infectieuses sur la santé mondiale
- **Identifier** les corrélations entre différents facteurs de santé
- **Comparer** les évolutions temporelles des indicateurs sanitaires
- **Mettre en évidence** les disparités géographiques et démographiques

## → Installation et Utilisation

### Prérequis
- Installer `Tableau Public`

### Étapes
1. Télécharger le fichier `projet.twbx`
2. Ouvrir avec Tableau Public
3. Explorer les différents dashboards via les onglets
4. Utiliser les filtres interactifs pour personnaliser les vues

---

##  Fonctionnalités Interactives

- **Filtres dynamiques** : Sélection par pays, année, genre
- **Cartes interactives** : Visualisation géographique avec gradients de couleur
- **Graphiques temporels** : Évolution des indicateurs dans le temps
- **Comparaisons** : Vues côte à côte pour analyses comparatives
- **Tooltips enrichis** : Informations détaillées au survol

---

##  Exemples de Visualisations

Le projet comprend notamment :
- Cartes choroplèthes pour la répartition géographique
- Graphiques en lignes pour les tendances temporelles
- Graphiques en barres pour les comparaisons
- Graphiques combinés pour les corrélations

##  Sources de Données

Les données proviennent d'un ensemble de données complet sur l'espérance de vie, couvrant :
- **Période** : Données historiques et récentes
- **Couverture** : Mondiale (nombreux pays)
- **Granularité** : Données annuelles par pays et par genre
